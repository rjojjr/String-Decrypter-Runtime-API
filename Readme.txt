The StringDecryptor library is made to decrypt strings at runtime,
that have been encrypted with SourceTools String Encryption function.

This API has one public class: Decryptor

Initialize:

To initialize a Decryptor object, call:

Decryptor.getInstance(String passphrase): Where passphrase is the 
	password used to encrypt the stringsin SourceTools.

or

Decryptor.getInstance(File key):  Where file key is the .key
	file automatically generated by SourceTools.

Use:

Simply wrap your strings in the <Decryptor Object>.decrypt(String string)
method.
